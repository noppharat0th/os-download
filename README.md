## ทำไมต้องใช้ OSDown?
การเตรียม USB Boot ทั่วไปมักจะยุ่งยาก ต้องโหลด Ventoy มาลงเอง แล้วต้องไปนั่งหาโหลดไฟล์ ISO (Windows, Linux ฯลฯ) จากหลายๆ เว็บ แอปนี้จะช่วย **Automate** ทุกอย่างให้ในที่เดียว

## Feature - ฟีเจอร์
- ติดตั้ง Ventoy ลง USB Flash Drive ได้ทันที
- มีรายการ ISO ยอดนิยมให้เลือก ไม่ต้องเปิด Browser หาเอง

## Install - ติดตั้ง
1. ดาวน์โหลดเวอร์ชันล่าสุดจาก [Releases](../../releases)
2. เปิดไฟล์ `OSDown.exe`
3. เสียบ USB แล้วเริ่มใช้งานได้เลย!

## How it use - วิธีใช้งาน?
1. หน้าหลัก เลือกดาวโหลด file iso ตามรายการที่ต้องการ window linux.. หรือ ค้นหา OS ที่ต้องการได้
2. หน้าตั้งค่าสามารถ เลือกลง VENTOY ในแต่ละ USB ได้ และเลือก INSTALLATION OPTION
3. restart computer และ boot usb Ventoy และติดตั้งได้เลย
